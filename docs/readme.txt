Para ejecutar el proyecto localmente se deberá contar con node y angular cli en el equipo.

Para el caso de angular cli se puede seguir los pasos del siguiente link: https://cli.angular.io/

Para levantar localmente la aplicación: 

Back-node
•	Se deberá ir a la carpeta back-node donde será necesario ejecutar desde esta posición npm install. 
•	Luego ejecutar node server.js 

Front-angular
•	Se deberá ir a la carpeta front-angular donde será necesario ejecutar npm install.
•	Luego con el comando ng serve se levantará localmente el proyecto
